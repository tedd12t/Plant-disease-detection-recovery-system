# Plant Disease Detection & Recovery System

A national multilingual plant disease detection and recovery system that leverages machine learning to identify plant diseases and provide recovery recommendations.

## Overview

This system is designed to assist farmers, agricultural experts, and plant enthusiasts in detecting plant diseases and providing actionable recovery solutions. The platform supports multiple languages for accessibility across diverse regions.

## Features

- 🌾 **Disease Detection**: Advanced ML model for accurate plant disease identification
- 🌍 **Multilingual Support**: Interface and documentation in multiple languages
- 💊 **Recovery Recommendations**: Personalized recovery strategies for detected diseases
- 📊 **Easy-to-Use Interface**: Jupyter Notebook-based analysis and reporting
- 🔬 **Data-Driven**: Based on comprehensive plant disease datasets

## Technology Stack

- **Jupyter Notebook** (85.6%) - Interactive analysis and model development
- **Python** (14.4%) - Core backend logic and utilities

## Installation

### Prerequisites
- Python 3.8 or higher
- Jupyter Notebook
- Required dependencies (see `requirements.txt`)

### Setup

```bash
# Clone the repository
git clone https://github.com/tedd12t/Plant-disease-detection-recovery-system.git
cd Plant-disease-detection-recovery-system

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook
```

## Usage

1. Open the Jupyter notebooks in the repository
2. Follow the step-by-step instructions for disease detection
3. Upload plant images or data
4. Receive disease diagnosis and recovery recommendations
